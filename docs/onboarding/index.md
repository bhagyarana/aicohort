# Onboarding & Setup

Welcome to the **Agentic AI Training Program**! This comprehensive course will take you from the fundamentals of AI agents to building production-ready systems.

## What You'll Learn

By the end of this training, you will be able to:

- **Build intelligent AI agents** using LangChain and LangGraph
- **Implement RAG pipelines** for knowledge-augmented AI systems
- **Create multi-agent systems** that collaborate on complex tasks
- **Deploy production-ready agents** with proper monitoring and testing

## Training Structure

```
Week 1-2: Foundations
├── Module 1: LangChain Fundamentals
└── Module 2: Advanced LangChain (RAG, Chains)

Week 3-4: Agent Development
├── Module 3: LangGraph Essentials
├── Module 4: Agents, Tools & MCP
└── Module 5: Production Patterns

Week 5-6: Implementation
├── Single Agent Patterns (3 implementations)
└── Capstone Project
```

## Quick Start

If you already have Python and your environment set up, you can jump straight to:

```bash
# Clone the training repository
git clone https://github.com/aicohort/agentic-ai-training.git
cd agentic-ai-training

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```
