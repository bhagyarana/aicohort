# Learning Modules

The Agentic AI Training consists of 5 comprehensive modules, each building on the previous to take you from fundamentals to production-ready AI agents.

## Module Overview

| Module | Topic | Duration | Focus |
|--------|-------|----------|-------|
| **1** | [LangChain Fundamentals](/learn/modules/module-1) | 90 min | Core concepts, prompts, LCEL |
| **2** | [Advanced LangChain](/learn/modules/module-2) | 90 min | Embeddings, RAG, chains |
| **3** | [LangGraph Essentials](/learn/modules/module-3) | 90 min | Stateful agents, workflows |
| **4** | [Agents, Tools & MCP](/learn/modules/module-4) | 90 min | Tool use, multi-agent |
| **5** | [Production Patterns](/learn/modules/module-5) | 90 min | Deployment, monitoring |

## Module Structure

Each module follows a consistent structure:

### 1. Overview (30 minutes)
- **Theory & Concepts**: Understand the why before the how
- **Architecture**: System design and patterns
- **Use Cases**: Real-world applications

### 2. Hands-on (60 minutes)
- **Guided Labs**: Step-by-step Jupyter notebooks
- **Code Examples**: Production-quality samples
- **Exercises**: Practice problems to solidify learning

### 3. Resources
- **Documentation Links**: Official docs and guides
- **Further Reading**: Deep dives into topics
- **Sample Projects**: Reference implementations

## Key Technologies

Throughout the modules, you'll work with:

### LLM Providers
- **OpenAI** - GPT-4, GPT-4o
- **Google AI** - Gemini 1.5
- **Groq** - Llama 3, Mixtral
- **Anthropic** - Claude 3

### Frameworks
- **LangChain** - LLM application framework
- **LangGraph** - Stateful agent workflows
- **ChromaDB** - Vector storage
- **FastAPI** - API deployment

### Tools & Protocols
- **MCP** - Model Context Protocol
- **Tavily** - Web search
- **Custom Tools** - Function calling
